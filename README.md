
# ESCAPE THE DUNGEON - DUNGEON CRAWLER JAM

A brief description of what this project does and who it's for

You are trapped inside a dark dungeon, will you find the exit and survive? or perish inside the dungeon?

CONTROLS

    MOVEMENT: WASD
    ROTATE VIEW: QE
    ATTACK: J
    HEAL: K
